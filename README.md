# INTERACTIVE DATABASE MANAGEMENT WITH PYTHON 🌿🧬

Professional database management system with integrated genomic analysis, developed as part of the **MSc IN BIOINFORMATICS**.

## 📋 Project Overview
This repository showcases a custom botanical database designed to bridge the gap between laboratory documentation and computational biology. The system ensures high standards of data integrity while providing advanced tools for DNA sequence processing and protein analysis.

## 🚀 Technical Highlights
* **Genomic Integration:** Capacity to store and validate DNA sequences for each botanical entry.
* **Bioinformatics Processing:** Automated DNA transcription and translation into protein sequences using **Biopython**.
* **Advanced Sequence Search:** Substring matching algorithm to identify plants containing specific genetic motifs.
* **Data Visualization:** Automated generation of proteomic profiles (amino acid frequency histograms) using **Matplotlib**.
* **Software Engineering:** Modular architecture with robust error handling and secure user interaction.
* **Smart Interaction:** Implementation of fuzzy string matching (**difflib**) and Regex for biological nomenclature cleaning.

## 🛠️ Technologies & Libraries
* **Language:** Python 3.x
* **Bioinformatics:** `Biopython` (Bio.Seq)
* **Visualization:** `Matplotlib`
* **Core Utilities:** `json` (Data Persistence), `re` (Regex), `difflib` (Fuzzy Matching), `collections.Counter`.

## 📁 Main Features 
1. **DNA Storage:** Structured storage of genetic sequences linked to botanical metadata.
2. **Genetic Query System:** Search functionality based on DNA sequence patterns.
3. **Molecular Workbench:** Real-time transcription (DNA to mRNA) and translation (mRNA to Protein).
4. **Proteomic Profiling:** Statistical visualization of amino acid distribution.
