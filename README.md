# INTERACTIVE DATABASE MANAGEMENT WITH PYTHON 🌿🧬

Professional database management system developed as part of the **MSc IN BIOINFORMATICS**.

# Project Overview
This repository showcases the design and implementation of a custom botanical database. It bridges the gap between traditional laboratory documentation and computational data management, ensuring high standards of data integrity and reproducibility.

# Technical Highlights
**Software Engineering:** Built with a focus on modularity and secure user interaction.
**Data Persistence:** Implemented structured storage using JSON for seamless data recovery.
**Fuzzy Matching:** Utilizes the `difflib` library to handle user input errors.
**Input Validation:** Employs Regular Expressions (**Regex**) for nomenclature cleaning.

# Technologies
**Language:** Python 3.x
**Core Libraries:** `json`, `re`, `difflib`, `os`.
